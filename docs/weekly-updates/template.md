## 🚦 Status Updates

A **traffic light system** so teammates get a quick understanding of each team's situation:

- 🔴 **very bad** - big problem / need help urgently
- 🟡 **not ideal** - some resolvable issues, might need help
- 🟢 **all great** - no problems, ideal circumstances

Each team fills in one row. Keep entries concise — bullet points are fine.

> **How to fill this in:**
> - **Overall**: pick one emoji (🔴 / 🟡 / 🟢) that summarizes your week
> - **Progress**: what got *done*. Tasks not finished should be **noted explicitly with an estimate of how long they'll take** (e.g. *"~2 days left"*)
> - **Challenges**: blockers, personal, team, or other issues that interfered with completion
