Thesis Discovery Platform API (0.1.0)

Download OpenAPI specification:

Simplified API contract for the thesis discovery platform.

The system has four services:

  • Thesis Service: owns PostgreSQL data and exposes the frontend API.
  • Scraping Service: periodically scrapes chair/source websites and sends extracted results to the Thesis Service.
  • GenAI Service: converts raw scraped content into structured thesis entries.
  • Vector Search Service: owns the vector database and supports semantic search plus reindexing.

Important design decision: During every successful scrape run for a chair, the Thesis Service replaces all existing thesis proposals for that chair in one transaction before instructing the Vector Search Service to refresh the corresponding vector index entries.

Search and filter theses

Search thesis proposals using traditional filters, natural language, or both.

Request examples:

  • Only filters: degreeType = "MASTER", researchAreas = ["AI"]
  • Only natural language: "I want a thesis about semantic search with LLMs"
  • Hybrid: natural language query plus traditional filters

The Thesis Service fetches matching structured data from PostgreSQL. If naturalLanguageQuery is present, it also calls the Vector Search Service and combines vector results with relational filtering.

Replace all thesis proposals for a chair

Called by the Scraping Service only after successful scraping and successful GenAI extraction for one chair/source endpoint.

Thesis Service behavior:

  1. In one transaction, delete all existing thesis proposals for chairId and insert the newly extracted thesis proposals.
  2. Commit only if the full replacement succeeds.
  3. After relational replacement succeeds, call the Vector Search Service to replace the vector index entries for the same chair.

If validation or insertion fails, the old thesis data must remain unchanged.

path Parameters
sourceEndpointId
required
integer <int64>

Internal source endpoint id.

Request Body schema: application/json
required
required
Array of objects (ThesisProposalInput)

string or null

http://localhost:8082/internal/v1/thesis-service/source-endpoints/{sourceEndpointId}/theses
